# Меню для столовой «Фаберже»

## Описание проекта
Система отображения меню для киосков самообслуживания. Меню разделено на две страницы (левое и правое меню), которые автоматически обновляются каждые 30 секунд.

## Структура файлов
```
/workspace/
├── menu-left.html      # Левая страница меню (Салаты, Вторые блюда)
├── menu-right.html     # Правая страница меню (Первые блюда, Гарниры, Напитки)
├── menu.csv            # Файл с данными меню
├── scripts.js          # JavaScript для загрузки и отображения меню
├── styles.css          # Стили оформления
└── README.md           # Этот файл
```

## Формат CSV файла
Файл `menu.csv` имеет следующую структуру:

```
цель;Категория;Наименование;Порция;Цена
```

### Поля:
- **цель** — тип карточки: `комбо`, `набор 1`, `набор 2`, `набор 3` или пусто (для обычных блюд)
- **Категория** — одна из: `Салаты`, `Первые блюда`, `Вторые блюда`, `Гарниры`, `Напитки`
- **Наименование** — название блюда
- **Порция** — вес/объём (например, `1/100`, `1/350`, `1 ст.`)
- **Цена** — цена в рублях (число)

### Примеры:
```csv
цель;Категория;Наименование;Порция;Цена
комбо;Салаты;ОЛИВЬЕ;1/100;180
набор 1;Салаты;МОРКОВЬ С ЧЕСНОКОМ;1/100;120
;Салаты;ВИНЕГРЕТ;1/100;120
;Первые блюда;БУЛЬОН КУРИНЫЙ;1/350;150
набор 2;Вторые блюда;РЫБА ЗАПЕЧЁННАЯ;1/130;220
```

### Логика работы с поле "цель":
- Если поле пустое — блюдо отображается только в списке категории
- Если указано `комбо` — создаётся карточка КОМБО с этим блюдом
- Если указано `набор 1/2/3` — создаётся карточка НАБОР №1/2/3
- Все блюда с одинаковой целью группируются в одну карточку
- Цена карточки = сумма цен всех входящих блюд

## Запуск локального сервера

### Python 3:
```bash
cd /workspace
python3 -m http.server 8000
```

### После запуска откройте в браузере:
- Левое меню: http://localhost:8000/menu-left.html
- Правое меню: http://localhost:8000/menu-right.html

## Обновление меню
1. Откройте файл `menu.csv`
2. Добавьте/измените/удалите строки с блюдами
3. Сохраните файл
4. Обновите страницу в браузере (или дождитесь автообновления через 30 сек)

## Примечания
- Страницы автоматически обновляются каждые 30 секунд
- Часы показывают текущее время и обновляются каждую минуту
- Карточки комбо/наборов генерируются автоматически на основе данных CSV
- Максимум может быть 3 набора + 1 комбо одновременно
